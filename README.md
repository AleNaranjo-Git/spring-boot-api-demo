# Spring Boot JWT Authorization Demo

This project demonstrates JWT-based authentication and role-based authorization using OIDC-style ID and Access tokens.

## Features
- Two users with different roles
- Two protected API resources
- Authorization based on JWT role claims
- Works locally and in the cloud [Website](https://spring-boot-api-demo-production.up.railway.app/)

## Run
* Run locally `./mvnw spring-boot:run`
* For a demo, open `/demo` in the [Browser](https://spring-boot-api-demo-production.up.railway.app/)
* Use `/token` and curl to test authorization
